# TargetingListType (enum)
TargetListTypeは、ターゲットリストの種類を表示します。

### Service
+ [RetargetingListService](../services/RetargetingListService.md)

| 値 | 説明 | 
|---|---|
| DEFAULT_LIST| デフォルトリストです。 |
| RULE| ルールタイプのリストです。 |
| COMBINATION| 組み合わせタイプのリストです。 |
| SIMILARITY| 類似ユーザリストです。 |
