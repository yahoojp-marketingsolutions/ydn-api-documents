# TargetListDeliverType (enum)
TargetListDeliverTypeは、ターゲットリスト種別（配信対象・配信除外対象の設定）を表します。
### Service
+ [AdGroupTargetService](../services/AdGroupTargetService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| INCLUDE| string| 配信対象です。 |
| EXCLUDE| string| 配信除外です。 |
