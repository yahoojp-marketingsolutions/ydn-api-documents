# DictionaryService
DictionaryServiceは、審査否認理由、地域情報、インタレストカテゴリー、サイトカテゴリーの一覧を提供します。

#### WSDL
| environment | url |
|---|---|
| production  | https://location.im.yahooapis.jp/services/V5.0/DictionaryService?wsdl|
| sandbox  | https://sandbox.im.yahooapis.jp/services/V5.0/DictionaryService?wsdl|
#### Namespace
http://im.yahooapis.jp/V5
#### サービス概要
各種情報の一覧を取得します。
#### 操作
DictionaryServiceで提供される操作を説明します。
