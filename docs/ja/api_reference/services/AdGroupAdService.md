# AdGroupAdService
AdGroupAdServiceは広告の操作を提供します。
#### WSDL
| environment | url |
|---|---|
| production  | https://location.im.yahooapis.jp/services/V5.1/AdGroupAdService?wsdl|
| sandbox  | https://sandbox.im.yahooapis.jp/services/V5.1/AdGroupAdService?wsdl|
#### Namespace
http://im.yahooapis.jp/V5
#### サービス概要
広告の取得と更新を行います。
#### 操作
AdGroupAdServiceで提供される操作を説明します。
