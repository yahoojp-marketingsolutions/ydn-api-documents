# ConversionTrackerService
ConversionTrackerServiceでは、コンバージョントラッカー情報の取得および追加・更新を行います。

#### WSDL
| environment | url |
|---|---|
| production  | https://im.yahooapis.jp/services/V5.0/ConversionTrackerService?wsdl|
| sandbox  | https://sandbox.im.yahooapis.jp/services/V5.0/ConversionTrackerService?wsdl|
#### Namespace
http://im.yahooapis.jp/V5
#### サービス概要
コンバージョントラッカー情報の取得およびコンバージョントラッカー情報の追加・更新を行います。
#### 操作
ConversionTrackerServiceで提供される操作を説明します。
