# AdGroupTargetService
ターゲティング設定に関する情報の取得および更新を行います。
#### WSDL
| environment | url |
|---|---|
| production  | https://location.im.yahooapis.jp/services/V5.0/AdGroupTargetService?wsdl|
| sandbox  | https://sandbox.im.yahooapis.jp/services/V5.0/AdGroupTargetService?wsdl|
#### Namespace
http://im.yahooapis.jp/V5
#### サービス概要
広告グループのターゲティング設定を行います。
#### 操作
AdGroupTargetServiceで提供される操作を説明します。
