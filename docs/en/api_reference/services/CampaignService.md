# CampaignService
Use this service to get, add, update, or delete campaigns.
#### WSDL
| environment | url |
|---|---|
| production  | https://location.im.yahooapis.jp/services/V5.0/CampaignService?wsdl|
| sandbox  | https://sandbox.im.yahooapis.jp/services/V5.0/CampaignService?wsdl|
#### Namespace
http://im.yahooapis.jp/V5
#### Service Overview
Offers campaign operation.
#### Operation
Describes operations provided by CampaignService.
## get
